## Key Takeaways

1. **Servlet Life Cycle**: init() â†’ service() â†’ destroy()
2. **One Servlet Instance**: Container creates one instance per servlet class
3. **Multiple Threads**: Each request handled in separate thread
4. **URL Mapping**: web.xml maps URLs to servlet classes
5. **Request/Response**: Container creates new objects for each request
6. **Form Processing**: POST data extracted via request.getParameter()
7. **Headers**: Accessed through request.getHeader() and getHeaderNames()
